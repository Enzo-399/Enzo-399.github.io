下面是激活教程：

1. 点 Help。

2. 点 Enter License。

3. 复制上面的 License 并输入。

4. 显示 Thanks for purchasing! 就是激活成功了。

5. 最后同第2步，不过是点 About Sublime Text 查看版本以及激活信息。
